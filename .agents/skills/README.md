# Codex Skills Index (Cloudflare)
- **Purpose**: Quick map of Cloudflare-specialist skills available in this repo.
- **MCP**: Cloudflare Documentation / Bindings / Observability servers are available; skills call them as needed.

## Skills
- **cloudflare-docs-specialist** — Search/quote official Cloudflare docs via MCP; prefer authoritative limits, flags, config keys.
- **workers-specialist** — Workers runtime + Hono patterns, bindings, security/perf for this codebase.
- **workers-ai-specialist** — Model selection, embeddings, RAG, AI Gateway, caching; remote-only AI/Vectorize.
- **observability-specialist** — Structured JSON logging, tracing (5% sampler), metrics/dashboards/alerts setup.
- **wrangler-workflows** — wrangler dev (remote), deploy, migrations, secrets, bindings validation.

## Usage
1) When a request matches a skill description, load that `SKILL.md` for guidance.
2) Prefer MCP doc searches for up-to-date Cloudflare behavior; cite official pages.
3) Keep answers concise; link to GitHub issues/PRs for history (#12–#19, #26–#27).
